# video-editor

Darshan's portfolio (Option B - light theme) — static single-file site.

Files:
- `index.html` — main site (booking form, portfolio placeholders, client galleries, OTP, WhatsApp share).
- `assets/` — place your images/videos here and update `index.html` paths accordingly.

How to use:
1. Edit `index.html` to replace placeholders with your images/videos and to customize text/prices/passwords.
2. Deploy to GitHub Pages or Vercel.

Vercel deployment (recommended):
- Import this repository into Vercel (New Project → Import Git Repository). No build command required.
